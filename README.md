# まいにち家計簿 v1.1

iPhoneで生活費をコントロールするための、個人用・静的Webアプリです。サーバー、ログイン、外部DB、外部ライブラリは使いません。

## 使い始める

`index.html` をブラウザで開くか、GitHub Pagesで公開して使います。支出や設定は、開いたブラウザの `localStorage` に保存されます。

最初に「設定」で生活費・家飲み・タバコ予算を確認し、「入力」から支出を登録してください。登録後は、同じ内容をクイック入力として保存できます。

ChatGPTでレシートや利用明細を読み取った場合は、「入力」→「ChatGPTの読み取り結果を取り込む」からCSV形式の結果を貼り付け、内容を確認してまとめて登録できます。

## GitHub Pagesで公開する

1. このフォルダのファイルをGitHubリポジトリの公開ブランチへ保存します。
2. GitHubのリポジトリで `Settings` → `Pages` を開きます。
3. `Deploy from a branch` を選び、公開ブランチ（通常は `main`）と `/ (root)` を指定して保存します。
4. 表示されたURLをiPhoneのSafariで開きます。
5. Safariの共有ボタンから「ホーム画面に追加」すると、アプリのように起動できます。

## データについて

- データは端末・ブラウザごとに保存され、端末間で自動同期されません。
- Safariの履歴・Webサイトデータを消すと、家計簿データも消える場合があります。
- 定期的に「設定」→「JSONバックアップ」を保存してください。
- CSVは明細の持ち出し用、JSONはカテゴリや予算を含む完全バックアップ用です。

## ファイル

- `index.html`: 4画面と各モーダルの構造
- `style.css`: iPhone向けレイアウトとデザイン
- `app.js`: 保存、集計、編集、CSV、バックアップ処理
- `tests/smoke-test.mjs`: 主要操作の自動スモークテスト
